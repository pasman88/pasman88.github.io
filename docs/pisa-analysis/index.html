Analisi dei dati PISA 2018 con R

Questa guida descrive come analizzare i dati PISA utilizzando R.

Sono presentati due approcci:

  • Implementazione manuale completa
  • Utilizzo del pacchetto EdSurvey

Installazione librerie



install.packages("tidyverse")
install.packages("lme4")
install.packages("survey")
install.packages("mitools")
install.packages("EdSurvey")

library(tidyverse)
library(lme4)
library(survey)
library(mitools)
library(EdSurvey)

Analisi manuale (PV + BRR + Rubin)

Questo approccio implementa completamente la metodologia PISA. 



student <- read.csv("CY07_MSU_STU_QQQ.csv")
school  <- read.csv("CY07_MSU_SCH_QQQ.csv")

data <- student %>%
left_join(school, by=c("CNT","CNTSCHID"))




estimate_model <- function(data,outcome){

formula_model <- as.formula(
paste(outcome,
"~ ESCS + ST004D01T + IMMIG +
(1|CNTSCHID) + (1|CNT)")
)

model <- lmer(formula_model,data=data)

fixef(model)

}

Analisi con EdSurvey

Il pacchetto EdSurvey automatizza:

  • Plausible Values
  • replicate weights
  • regole di Rubin


pisa <- readPISA(
path="PISA2018/",
database="INT"
)

model <- lm.sdf(
PVFLIT ~ ESCS + ST004D01T + IMMIG,
data=pisa
)

summary(model)
