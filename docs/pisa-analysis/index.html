Analisi dei dati PISA 2018 con R

Questa guida mostra come analizzare i dati dell'indagine OCSE PISA utilizzando R.

Sono presentati tre approcci:

  • Analisi manuale con Plausible Values
  • Analisi avanzata con replicate weights (BRR)
  • Utilizzo del pacchetto EdSurvey

3 Metodo EdSurvey

Il pacchetto EdSurvey automatizza la gestione di Plausible Values e replicate weights nelle indagini educative come PISA. 



install.packages("EdSurvey")

library(EdSurvey)

pisa <- readPISA(
path="PISA2018/",
database="INT"
)

model <- lm.sdf(
PVFLIT ~ ESCS + ST004D01T + IMMIG,
data=pisa
)

summary(model)
